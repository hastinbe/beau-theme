My custom theme.

![](example.png)

## Installation

Clone this repository: `git clone https://github.com/hastinbe/beau-theme.git ~/beau-theme`

#### Xdefaults

`#include "~/beau-theme/Xdefaults/beau"`

#### Xresources

`#include "~/beau-theme/Xresources/beau"`
